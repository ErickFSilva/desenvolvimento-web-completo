Destructuring

- Operador destructuring: é um operador de desestruturação retirando valores de dentro de uma estrutura. Essa estrutura pode ser um array ou um objeto.

Composição:
1. Quando criamos objetos a partir de objetos literais, funções construtoras ou classes temos uma "composição é um". Então aquele objeto é alguma coisa.
2. Mas quando existe uma relação de atribuição de um objeto como sendo o valor de algum atributo de outro objeto nós temos a "composição tem um".
Tokens:
1. Array => []
2. Objeto => {}
